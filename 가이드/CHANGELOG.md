# 변경 이력 (CHANGELOG)

> 세션별 완료 작업을 기록하는 변경 이력 문서입니다. 새 Claude 세션에서 이전 작업 내용을 빠르게 파악할 수 있도록 합니다.

---

## 작성 형식

각 세션은 아래 형식으로 기록합니다:

```
### 세션 N (YYYY-MM-DD)

#### #번호 작업제목
- 변경 내용 요약
- 수정 파일: `경로/파일명`
- 비고: (필요 시)
```

### 작성 규칙
- **세션 단위**: 하나의 Claude 대화 세션 = 하나의 세션 블록
- **작업 번호**: `#1`부터 순차 증가, 세션이 달라도 번호는 이어감
- **수정 파일**: 실제 변경된 파일 경로를 명시
- **간결하게**: 무엇을 했는지 핵심만 기록 (코드 전문 X)

---

## 이력

### 세션 1 (YYYY-MM-DD)

#### #1 프로젝트 초기 설정
- 폴더 구조 생성
- 기본 가이드 문서 배치
- 수정 파일: `CLAUDE.md`, `가이드/PROJECT_GUIDE.md`
